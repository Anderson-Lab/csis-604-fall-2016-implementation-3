    For this assignment you will experiment with iterative graph processing.

    Part 1 (20%)

    Download and install http://giraph.apache.org/, which is an open source implementation of Google's Pregel. Show that everything is working correctly, but running the SimpleShortestPathsComputation sample (http://giraph.apache.org/quick_start.html).

    Part 2 (80%)

    Now we are going to implement Page Rank on giraph! Page Rank is one of the most famous algorithm. Here is a walk through of the theory of the algorithm and it even includes an implementation in Python: http://www.ccs.northeastern.edu/home/daikeshi/notes/PageRank.pdf. Your task is to port this algorithm to work on giraph. Three is a partial tutorial that will help you start: http://giraph.apache.org/pagerank.html.
